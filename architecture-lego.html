Lego-like modular architecture

clausr.ai · snap & swap

Every layer is a replaceable brick behind a clean interface. Swap any component independently — the pipeline stays intact.
📄
Session Input Layer
File ingestion, OCR, chunking with spatial bounding boxes, full-text indexing.
💡 Business value: Accept any document your clients send — no more "please resend as PDF". Works with scanned papers, phone photos, native docs.
Swap file format
PDF · DOCX · Image (JPEG/PNG)
Add: TIFF · XLSX · HTML · EML
⚙️
Loading Layer (Doc Store)
How document chunks are stored, indexed, and retrieved at query time.
💡 Business value: Stay fully air-gapped with local SQLite, or scale to millions of docs with Pinecone/Weaviate. Pick the infra that matches your deployment — not the other way around.
Swap vector / doc db
SQLite + FTS5 (local)
Add: Pinecone · Weaviate · pgvector · Chroma
🔧
Pipeline Layer · clausr.ai core
Per-turn orchestration: context restore → citation load → chunk search → LLM execution → evaluation → stream.
💡 Business value: Never locked into one LLM provider. Optimize between cost (DeepSeek), quality (Claude), ecosystem (GPT), or data residency (local models). The pipeline orchestrates all of them equally.
Swap LLM provider
DeepSeek V4
Also: Claude · GPT-4o · Llama · local models
📚
Knowledge Layer (Regulations)
Regulation data: clause definitions, reference text, citation validation rules.
💡 Business value: Expand to new markets overnight. Already have a compliance framework? Plugin your own internal policy DB. One platform, unlimited domains — GDPR, HIPAA, SOC2, EU VWTA, UN R13.
Swap regulation DB
GDPR (built-in)
Add: HIPAA · SOC2 · ISO 27001 · custom policy
✅
Evaluation Layer
Confidence scoring, finding summarization, post-validation with citation safety nets.
💡 Business value: Calibrate strictness to your risk tolerance. Low-stakes audits → fully automatic. High-stakes compliance → crank confidence thresholds, add human-in-the-loop gates.
Swap validation logic
confidence algo + word-overlap heuristics
Customize: thresholds · scoring fn · HITL gates · rules engine
📊
Present Layer
Response assembly, citation badges, verdict computation, export generation.
💡 Business value: Deliver results however your stakeholders need them. DOCX for legal, HTML dashboards for management, JSON API for downstream systems — no one changes their tools.
Swap report format
DOCX · HTML (SSE stream)
Add: PDF · JSON API · embedded widget · email
