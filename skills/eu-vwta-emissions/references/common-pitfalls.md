# Common Pitfalls — EU VWTA Emissions

## WLTP vs NEDC
Always confirm which test cycle applies. R154 WLTP replaces NEDC for new type approvals from 2017. Legacy approvals may use NEDC correlations.

## OBD Requirements
OBD stage depends on vehicle category and registration date. Check applicability before testing. Euro 6d requires demo capability for OBD thresholds.

## Mass Derogation
CO2 limits depend on reference mass. Do not compare measured values against the standard 130 g/km without checking whether mass derogation applies. Derogation formula: CO2 limit = 130 + 0.0457 × (reference mass − 1,305) for masses above 1,305 kg.

## RDE Conformity
Real Driving Emissions (RDE) must be tested on public roads with PEMS equipment. RDE results are reported as conformity factors, not absolute limits. Euro 6d requires CF ≤ 1.0 for NOx.

## Test Fuel
Emissions certification requires specific reference fuels (e.g., E10 for petrol, B7 for diesel). Using standard market fuels invalidates the test.
