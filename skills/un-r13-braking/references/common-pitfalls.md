# Common Pitfalls — UN R13 Braking

## Test Conditions
Brake tests must be performed with the vehicle at laden mass. Unladen tests are not valid for type approval. Test track must be dry with a peak friction coefficient ≥ 0.9.

## ABS Verification
ABS functionality must be verified on low-friction surfaces (µ ≤ 0.4). Dry-asphalt-only testing is insufficient. Split-µ surfaces should also be tested for ABS brake force distribution.

## Brake Fade Validation
The Type-I fade test requires 20 snubs, not full stops. If the test driver performs full stops instead of snubs, brake temperatures will exceed legal limits and the test must be repeated.

## Regenerative Braking Interaction
For hybrid and electric vehicles, regenerative braking transitions (blending) must not cause ABS intervention thresholds to be exceeded. The transition from regen to friction braking must be seamless.

## Secondary Brake Dual-Circuit Verification
Some manufacturers test secondary brake performance on a single circuit but fail to verify that the remaining circuit provides adequate pressure. Both circuits must be independently verified.
